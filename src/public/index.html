URL Shortener Microservice

Pass a URL as a query string value to the field-name 'name' within https://shourl.herokuapp.com/?name= and you will receive a shortened version via JSON response.

Example usage:
https://shourl.herokuapp.com/?name=https://www.google.com

Using the shortened version (e.g. https://shourl.herokuapp.com/a2Bc) will redirect to the website originally passed.

Addresses passed must follow the format: "http(s)://www.domain_name.top_level_domain".
